# APF-Ed: Análisis de Puntos de Función Educativos

Propuesta metodológica que adapta la métrica **ISO/IEC 20926:2009 (IFPUG)** al diseño instruccional y la didáctica de la informática.

## 🚀 Resumen del Modelo
El modelo **APF-Ed** permite transformar la planificación docente de una estimación intuitiva a una proyección técnica cuantitativa.

### Fórmula Maestra de Tiempo ($T$)
$$T = \frac{PFD \times VAFed}{P}$$

* **PFD:** Puntos de Función Didácticos (Carga bruta).
* **VAFed:** Factor de Ajuste Educativo (Entorno e IA).
* **P:** Productividad del grupo (Sugerido: 4-5 PFD/h).

### Configuración del VAFed (Escala Inversa)
Para una correcta usabilidad docente, se utiliza una escala de **0 a 5**:
- **0 (Óptimo):** Sin impacto negativo en el aprendizaje.
- **5 (Crítico):** Impacto total, requiere máximo recargo de tiempo.
